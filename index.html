Projects

Age Gender detection system using CNN and RNN deep learning techniques

The Age, Gender, and Ethnicity Detection System is a deep learning-based framework developed to detect and classify age, gender, and ethnicity from real-world facial images. With the growing demand for such systems in various applications like airports, police stations, and personalized recommendation systems, this project aims to provide accurate and reliable results. The system utilizes CNN and RNN to extract facial features and train models for classification.

Analysis of Airline Passenger satisfaction

The goal is to understand the key factors that contribute to customer contentment. The analysis was performed using R Studio, employing machine learning algorithms and data visualizations to identify correlations between variables and passenger satisfaction. The study finds that factors such as inflight entertainment, ease of online booking, and travel type significantly impact passenger satisfaction. Linear Discriminant Analysis (LDA) and Quadratic Discriminant Analysis (QDA) were found to be the most accurate machine learning models for this dataset.

Zonal Accident Risk Prediction system

The Zonal Accident Risk Prediction project analyzed a dataset of road accidents to identify common patterns and features contributing to accidents. Models like linear regression and random forest classification were used to predict factors such as speed limits and fatalities. The project aimed to reduce accidents by implementing safety measures based on the findings. Limitations included the absence of predictors for road surface and weather conditions.

IU Commuter Pass

The IUCommuterPass (IUCP) web application facilitates carpooling among students at Indiana University-Purdue University Indianapolis (IUPUI). It provides a platform for riders and passengers to connect, reducing transportation costs and environmental impact. The user-friendly interface includes features like trip cost calculation, check-in system, and messaging. The project is significant in offering affordable transportation, reducing pollution and traffic congestion, and promoting social connections among students.

John snows cholera map visualization using D3.js

Dr. John Snow's map recreated the 1854 cholera epidemic in London, revealing patterns and mitigating its effects through visualization. Interactive visualizations included a timeline chart, map, and pie charts showcasing variations in deaths by age and gender. Discoveries highlighted higher death rates among the elderly and children, with contaminated water pumps identified as hotspots.
